# 📜 CHANGELOG

이 문서는 **Photo & Video Rename + Copy Tool**의 버전별 변경 이력을 정리한 기록입니다.  
주요 기능 추가, 설계 변경, 안정성 개선 사항을 중심으로 작성되었습니다.

---

## v6.3 (2025-12)

### 🎥 동영상 메타데이터 처리 강화 (MOV 최적화)
- 동영상 촬영 시간 추출 우선순위 개선
  1. QuickTime:CreateDate
  2. QuickTime:MediaCreateDate
  3. MediaCreateDate
  4. CreateDate
  5. LastWriteTime (Fallback)
- MOV 파일에서 잘못된 날짜로 정렬되는 문제 해결
- MP4 / WMV에도 동일 로직 적용

---

## v6.2 (2025-12)

### 🔑 ID 생성 로직 안정화 (해시 기반)
- `IMG_####` 형식 파일
  - 기존 컷 번호(####) 그대로 유지
- 그 외 파일
  - `파일명 + 파일크기` 기반 SHA-1 해시
  - 앞 6자리 사용 → 실행할 때마다 동일한 ID 생성
- 파일 처리 순서에 따라 ID가 달라지던 문제 해결
- 촬영 시간이 같은 파일 다수 존재 시 충돌/무한 루프 방지

### 🎥 동영상 확장자 추가
- MOV, WMV 지원 추가
- PHOTO / VIDEO 자동 분기 유지

# 📜 CHANGELOG

이 문서는 **Photo & Video Rename + Copy Tool**의 버전별 변경 이력을 정리한 기록입니다.

## v6.9 (2025-12)

### 🚀 실행 안정성 & 사용자 경험 개선
- **중단 복구(Resume) 기능 추가**
  - 이전 실행 로그를 기반으로 이미 처리된 파일 자동 스킵
  - 재실행 시 중단 지점 이후부터 안전하게 이어서 처리 가능
- 실행 종료 직전 오류를 방지하기 위한
  - MessageBox 출력 로직 안정화
  - 문자열 처리 및 enum 타입 사용 방식 정리

### ⏱️ 진행 정보 표시 강화
- 진행률 창에 추가 정보 표시
  - 전체 경과 시간 (Elapsed Time)
  - 남은 예상 시간 (ETA)
  - 평균 처리 속도 (files/sec)
- 대용량 작업 시 처리 시간 예측 가능

### ⚡ 성능 최적화
- 기존 HashSet 기반 파일 캐싱 로직과 Resume 기능 결합
- 이미 처리된 파일을 디스크 접근 없이 즉시 스킵
- 스킵 파일이 많은 환경에서도 체감 속도 개선

### 📖 문서 업데이트
- README / HowToUse 문서에 v6.9 기능 반영
- 중단 복구 및 진행 정보 표시 방식 상세 설명 추가
